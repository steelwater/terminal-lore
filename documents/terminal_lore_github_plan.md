# Terminal Universe – Project Plan

## Purpose

This repository is the umbrella project for the “Terminal Universe”:
A collection of computer-terminal inspired stories, experiments, ASCII art, timelines, and small systems.

It acts as:
- A creative archive
- A technical playground
- A narrative “world index” for related subprojects

## Current Structure

Subfolders represent different concepts and experiments:

- `ascii/`  
  ASCII art and text-based visuals

- `carl/`  
  Game/system experiments (zero-player, scripted agents)

- `terminal timeline/`  
  Timeline site or data for the terminal universe

- `terminal1/`  
  Early terminal experiments

- `terminal582/`  
  Specific story arc or visual concept

- `tpc/`  
  Experimental or unknown purpose (needs clarification or cleanup)

## Goals

- Make this repo the **index and lore hub** for the Terminal Universe
- Clearly document what each subproject is
- Prepare major subprojects for extraction into their own repos
